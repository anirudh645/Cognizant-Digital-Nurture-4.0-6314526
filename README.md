## How to Run

Navigate to the appropriate `Code` directory and use the following commands:

```sh
javac FileName.java
java FileName
```

Replace `FileName` with the main class you want to run (e.g., `FinancialForecasting`, `Test`, or `FactoryMethodTest`).
